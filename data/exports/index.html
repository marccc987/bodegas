📐 Metodología
  1. Selección de blancos: Se identificaron 5 cuentas que han hecho críticas o apoyos a la candidata Paloma Valencia y reciben avalanchas de respuestas negativas.
  2. Recolección via API: Se buscaron ~500 replies recientes a cada blanco usando X API v2 (modelo pay-per-use). Total: 2,547 tweets.
  3. Detección de coordinación: Se identificaron cuentas que aparecen respondiendo a 2 o más blancos distintos. Responder negativamente a múltiples críticos de forma consistente es un indicador fuerte de comportamiento coordinado.
  4. Perfilamiento: Se obtuvieron datos completos de cada cuenta (fecha de creación, followers, bio, avatar) para análisis adicional.

Nota: Este análisis identifica patrones de comportamiento coordinado. No todas las cuentas multi-blanco son necesariamente bots; algunas pueden ser usuarios reales muy activos políticamente.
